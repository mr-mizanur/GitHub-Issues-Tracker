# 1 What is the difference between var, let, and const?
Ans: Difference between var, let, and const:
var  :Function scoped, can be redeclared, and can be reassigned.
let  :Block scoped, cannot be redeclared, but can be reassigned.
const:Block scoped, cannot be redeclared, and cannot be reassigned.

# 3 What is the difference between map(), filter(), and forEach()?
Ans:difference between map(), filter(), and forEach():
map()    : Creates a new array, transforms each element.
filter() :Creates a new array, keeps elements that match a condition
forEach():Just loops through the array, does not return a new array
